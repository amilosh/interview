 - Интерпертируемые и компилируемые языки программирования -
Компилятор - программа, которая считывает текст, написанный человеком (программу) в числа, которые компьютер воспринимает как команды процессора.


Компилируемый язык.
Human-readable код компилируется в инструкции для целевой машины
  ?в байткод
  ?в набор шустнадцатиричны символов
и уже сам компьютер (целевая машина) ннапрямую считывает этот скомпилированный 
Программа выполняется быстрее.
Низкоуровневые - компилируемые; ффективность выше кроссплатформенности. Контроль аппаратного обеспечения (управление памятью и использование процессора).
 C, C++, Erlang, Haskell, Rust и Go.
Минусы. Для запска программы её нужно скомпилировать. Сложность отладки, ведь для тетирования изменения нужно компилировать заново. Платформо-зависимы - машинный код зависит от машины, на которой компилируется и исполняется программа.

Три этапа:
- проверка синтакиса перед компиляцией (front end) -> генерация промежуточного представления
- оптимизирует эо предсталение 
- компилирует в машинный код для конткреной машины (ARM, Sparc, x86_32, PowerPC)


Интерпретируемый язык.
Компилировать код не нужно. Прграмма запускается под управллением другой программы - интерпретатор (делает незаметную компияцию построчно).
Процесс написания и тестирования быстрее (нет необходимости в промежуточном шаге компиляции).
Благодаря JIT-компиляция - процесс интерпретации быстрый (перевод часто используемых последовательностей инструкции в машинный код).
PHP, Perl, Ruby и Pyt
Плюсы: независимость от платформы; рефлексия; динамическая типизация; меньший размер исполняемых файлов; динамические области видимости.


Байткод-язык.
Java, C#.
Сначала компиляция из программного кода в байткод - нобор инструкций (числовы кодов, констант и слок на память) для интерпретатора. Затем этот байткод передается в Java Virtual Machine (JVM) - интерпретатор байткода (компилированного программного кода). JVM интерпретирует код также, как и обычный интерпретатор.
Байткод - он оптимизирован. Работает быстрее, чем на обычных


Python.
Интерпретатор CPython - интерпретирует программный код спо строкам сверу вниз в байт-код. Встроенная виртуальная машина выполняет тот байт-код.
